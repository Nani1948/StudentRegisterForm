STUDENT REGISTRATION SYSTEM

Project Description:
The Student Registration System is a simple web application developed using HTML ,CSS,Javascript.It allows user to registered student details,edit records and delete records.  The application uses the local storage to save the data/


Features:
1)Register new student
2)Edit existing records
3)Delete student recors
4)Store data using local storage
5)Form Validation:
   -Name should always accept only letters
   -ID accept only number
   -Valid email address
   -Contact accept at least 10 digit.
6)Prevent empty submission
7)verticall scrollbar
8)Responsive design for mobile ,tablet, desktop

Technologies:
-HTML
-CSS
-Javascript
-Local storage

How to run project:
1)Download or clone the repository
2)Open the project folder
3)Double click index.html.
4)Register student details using the form.
5)view,edit, and delete records as needed

Data Storage:
The application uses the browser's local storage to save student records
